Simple Selectors

These selectors target elements based on their most fundamental characteristics.

Element Selector

Selects all elements of a given HTML tag name.

Class Selector

Selects all elements with a specific attribute. It is denoted by a period () followed by the class name.

ID Selector

Selects a single, unique element with a specific attribute. It is denoted by a hash () followed by the ID name. An ID should only be used once per page.

Universal Selector

Universal Selector: Selects all elements on the page. It is denoted by an asterisk ().

Grouping Selector

Combines multiple selectors into a comma-separated list to apply the same styles to all of them, avoiding redundant code.

Combinator Selectors

Combinators are used to select elements based on the relationship between them and other elements in the document tree.

Descendant Selectors

Selects all elements that are a descendant of a specified element, regardless of how deep they are nested. It is denoted by a space between selectors.

Child Selector

Selects elements that are direct children of a specified element. It is denoted by the greater-than sign () (e.g., selects only elements whose immediate parent is a ).

Adjacent Sibling Selector:

Selects an element that is the immediate next sibling of another specified element. It is denoted by a plus sign () (e.g., selects the first element that immediately follows an )

General Sibling Selector

Selects all sibling elements that follow a specified element. It is denoted by a tilde () (e.g., selects all elements that follow an , as long as they share the same parent). [8, 13, 14, 15, 16]

Pseudo Sectors

Pseudo-class Selectors

Pseudo-classes select elements based on a specific state, condition, or position that is not described in the document tree. They are indicated by a single colon ()

  • :hover: Styles an element when the user's mouse cursor is over it (e.g., changes the style of a link when hovered).
  • :focus: Styles an element when it has focus (e.g., a form input field).
  • :nth-child(): Selects elements based on their position among a group of siblings (e.g., selects the second paragraph).
  • :visited:Styles links that the user has already visited (e.g., ).

Pseudo-element Selectors

Pseudo-elements are used to style a specific part of an element or to insert content before or after an element. They are indicated by a double colon ().

  • ::before: Inserts generated content before the actual content of an element
  • ::after: Inserts generated content after the actual content of an element
  • ::first-line: Styles the first formatted line of a block-level element
  • ::first-letter: Styles the first letter of a block-level element.

Attribute Selectors

These selectors select elements based on the presence or value of a specific HTML attribute. They are contained within square brackets ().

  • [attribute]: Selects all elements that have the specified attribute, regardless of its value (e.g., selects all anchor tags with a attribute)
  • [attribute="value"]: Selects elements where the attribute's value matches exactly (e.g., selects all input fields with the type "text").
  • [attribute^="value"]: Selects elements whose attribute value begins with a specific substring (e.g., selects links where the starts with "https")
  • [attribute$="value"]: Selects elements whose attribute value ends with a specific substring (e.g., `a[href$=".pdf"]href` ends with ".pdf")
