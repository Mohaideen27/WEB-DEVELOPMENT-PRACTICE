Understanding Boiler plate code

!DOCTYPE html : Informs the browser that this is an HTML5 document.

html lang="en" : The root element, defining the language.

meta charset="UTF-8": Sets the character encoding to UTF-8, supporting most characters.

meta name="viewport": Ensures the website is responsive on mobile devices.

title: Sets the text displayed on the browser tab.

body: Contains all visible content (text, images, links).
